# Diwali Sales Analysis – Python

This project focuses on analyzing Diwali sales data to uncover customer purchasing behavior and sales performance insights using Python.

The objective of this analysis is to understand customer demographics, identify purchasing trends, and determine top-performing products and categories to support data-driven business decisions.

## Key Highlights
• Performed data cleaning and preprocessing to prepare raw sales data for analysis  
• Conducted Exploratory Data Analysis (EDA) using Pandas, Matplotlib, and Seaborn  
• Analyzed customer purchasing behavior across states, age groups, gender, and occupations  
• Identified top-selling product categories and products to support demand forecasting and inventory planning  

## Tools & Technologies
• Python  
• Pandas  
• NumPy  
• Matplotlib  
• Seaborn  
• Jupyter Notebook  

## Dataset
• Diwali_Sales_Data.csv – Raw sales dataset used for analysis  

## Files Included
• Diwali_Sales_Analysis.ipynb – Jupyter Notebook containing data cleaning, EDA, and visualizations  
• Diwali_Sales_Data.csv – Sales dataset  
• README.md – Project documentation  

## Outcome
The insights generated from this analysis help businesses improve customer targeting, optimize inventory management, and enhance overall sales strategy by understanding purchasing patterns during the Diwali season.
