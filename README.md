# Diwali Sales Analysis – Python

This project focuses on analyzing Diwali sales data to uncover customer behavior patterns and sales insights using Python.

The analysis helps understand purchasing trends across different demographics and identifies high-performing products and categories to support data-driven business decisions.

Key Highlights:
• Performed data cleaning and preprocessing to prepare raw data for analysis
• Conducted Exploratory Data Analysis (EDA) using Python libraries such as Pandas, Matplotlib, and Seaborn
• Analyzed customer purchasing behavior across states, age groups, gender, and occupations
• Identified top-selling product categories and products to support demand forecasting and inventory planning

Tools & Technologies:
• Python
• Pandas
• Matplotlib
• Seaborn
• Jupyter Notebook

Outcome:
The insights generated from this analysis can help businesses improve customer targeting, optimize inventory management, and enhance overall sales strategy.
